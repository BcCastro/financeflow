# FinanceFlow - Dashboard de Finanzas Personales 🚀

FinanceFlow es una aplicación web responsiva diseñada para que los usuarios puedan gestionar y visualizar sus finanzas personales de forma intuitiva. Este proyecto fue construido con un enfoque profesional, optimizando la arquitectura del estado y garantizando una experiencia de usuario fluida.

## ✨ Características Principales
- **Control Total:** Registro ágil de ingresos y egresos clasificados por categorías.
- **Persistencia de Datos:** Uso de `localStorage` para salvaguardar la información del usuario de manera local sin necesidad de base de datos externa.
- **Diseño Mobile-First & Responsive:** Interfaz moderna y adaptada a cualquier tamaño de pantalla gracias a Tailwind CSS.
- **Arquitectura Limpia:** Control del estado global mediante React Context API, evitando el *prop drilling*.

## 🛠️ Tecnologías Utilizadas
- **React.js** (Vite para el empaquetado ultra rápido)
- **Tailwind CSS** (Estilos y Grid Responsivo)
- **Lucide React** (Paquete de iconos vectoriales modernos)
